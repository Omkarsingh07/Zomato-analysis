# Zomato Data Analysis Project

## Project Overview
This project focuses on analyzing Zomato restaurant data using Python and Tableau to uncover customer ordering behavior, restaurant trends, ratings, spending patterns, and online ordering insights.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights
- Tableau Dashboard Creation

# Business Questions Solved

## 1. What type of restaurant do the majority of customers order from?

Analyzed the most popular restaurant categories using count plots and bar charts.

---

## 2. How many votes has each type of restaurant received from customers?

Performed vote analysis to identify customer engagement across restaurant categories.

---

## 3. What ratings do the majority of restaurants receive?

Analyzed restaurant rating distribution using histograms and boxplots.

---

## 4. What is the average spending on each order?

Performed cost analysis using the approximate cost for two people column.

---

## 5. Which mode (online or offline) has received the maximum rating?

Compared online and offline restaurant ratings using boxplots.

---

## 6. Which type of restaurant receives more online orders?

Created heatmaps to analyze restaurant categories with higher online ordering activity.

---

# Key Insights

- Delivery restaurants are the most common restaurant type.
- Online ordering is highly popular among customers.
- Most restaurants fall within a medium price range.
- Restaurants with higher ratings generally receive more votes.
- Delivery and dine-out categories dominate customer engagement.
